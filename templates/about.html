Introduction

Weather Checker is my homework to my Python Course Python与深度学习基础.

How It Works

I used the Flask framwork to build the website. The data comes from a free API provided by OpenWeatherMap,so remember to connect to the Internet while viewning it.
The webpage framework is Bootstrap, and the template comes from 模板之家.
